Mocreak is an office enhancement tool that automatically downloads, installs, and deploys genuine Windows and Office with one click. This tool is completely free, ad-free, green, non-toxic, simple, efficient and safe.

### ✨ Software Features
- Brand new modern graphical interface. Supports light/dark two-color themes and adapts to languages ​​in multiple countries/regions to help you improve deployment efficiency.
![Feedback](images/feedback.png)
- A complete feedback system helps you locate and find problems encountered during deployment, and respond to solutions as soon as possible.

    #### 🪟 Windows
    ![Windows](images/windows.png)
    - Customizable installation options for various distributions of Windows 11 x64.
    -Support languages ​​from multiple countries or regions as the default language for new systems.
    - Supports "online download and installation" and "offline installation" Windows operating systems.
    - The system custom components that need to be installed can be highly customized, and you can even choose not to install Windows Defender, Notepad, etc.
    - Based on the native compilation mode, construct a Windows operating system that adapts to your computer on demand, greatly improving system stability and compatibility.
    - Except for necessary restarts during Windows installation, you can continue to use your computer for work.
    - For computers that do not meet the hardware configuration of TPM, CPU, memory, etc., implement compatibility policies to ensure smooth installation.
    - After installation, there is no need to configure complex computer settings and you can directly log in to the computer using a local account.

    #### 🏢Office
    ![Office](images/office.png)
    - Quickly download, install, and deploy the latest Office 2016, 2019, 2021, and 2024 versions of software with one click.
    - Supports installing Office to other partitions, including solid state drives, mechanical disks, removable disks, flash disks, memory disks, etc.
    - Word, PPT, Excel, Outlook, OneNote, Access, Visio, Project, OneDrive, Lync/Skype and other components can be installed in any combination.
    - Support languages ​​from multiple countries or regions as the Office interface language.
    - Before installing Office, the software automatically detects conflicting Office components and uninstalls them.
    - Supports "online download and installation" and "offline installation" dual strategies to install Office components.
    - You can use this tool for "Completely Remove Office Components" scenarios to completely remove Office when it is no longer needed.

### 🚀 Run & Test
- To ensure stable operation, this software supports running under the x64 system architecture of Windows 10 and above.
- The software has been tested on x64 systems of multiple Windows system versions and runs normally and stably.
