Mocreak は、ワンクリックで正規の Windows と Office を自動的にダウンロード、インストール、展開するオフィス拡張ツールです。このツールは完全に無料、広告なし、グリーン、無毒、シンプル、効率的、安全です。

### ✨ ソフトウェアの機能
- 真新しいモダンなグラフィカルインターフェイス。明/暗の 2 色のテーマをサポートし、複数の国/地域の言語に適応して導入効率の向上に役立ちます。
![フィードバック](images/フィードバック.png)
- 完全なフィードバック システムは、展開中に発生した問題を特定して発見し、できるだけ早く解決策に対応するのに役立ちます。

    #### 🪟 Windows
    ![Windows](images/windows.png)
    - Windows 11 x64 のさまざまなディストリビューション向けにカスタマイズ可能なインストール オプション。
    -新しいシステムのデフォルト言語として、複数の国または地域の言語をサポートします。
    - Windows オペレーティング システムの「オンライン ダウンロードとインストール」および「オフライン インストール」をサポートします。
    - インストールする必要があるシステム カスタム コンポーネントは高度にカスタマイズでき、Windows Defender、メモ帳などをインストールしないことも選択できます。
    - ネイティブ コンパイル モードに基づいて、オンデマンドでコンピュータに適応する Windows オペレーティング システムを構築し、システムの安定性と互換性を大幅に向上させます。
    - Windows のインストール中に必要な再起動を除いて、コンピュータを仕事に使い続けることができます。
    - TPM、CPU、メモリなどのハードウェア構成を満たさないコンピューターの場合は、スムーズなインストールを確保するために互換性ポリシーを実装します。
    ・インストール後は複雑なパソコンの設定が不要で、ローカルアカウントで直接パソコンにログインできます。

    #### 🏢オフィス
    ![オフィス](images/office.png)
    - 最新の Office 2016、2019、2021、2024 バージョンのソフトウェアをワンクリックですばやくダウンロード、インストール、展開できます。
    - ソリッド ステート ドライブ、メカニカル ディスク、リムーバブル ディスク、フラッシュ ディスク、メモリ ディスクなどの他のパーティションへの Office のインストールをサポートします。
    - Word、PPT、Excel、Outlook、OneNote、Access、Visio、Project、OneDrive、Lync/Skype およびその他のコンポーネントを任意の組み合わせでインストールできます。
    - Office インターフェイス言語として複数の国または地域の言語をサポートします。
    - Office をインストールする前に、ソフトウェアは競合する Office コンポーネントを自動的に検出し、アンインストールします。
    - Office コンポーネントをインストールするための「オンライン ダウンロードとインストール」と「オフライン インストール」の 2 つの戦略をサポートします。
    - このツールを「Office コンポーネントの完全な削除」シナリオに使用すると、不要になった Office を完全に削除できます。

### 🚀 実行とテスト
- 安定した動作を確保するために、このソフトウェアは Windows 10 以降の x64 システム アーキテクチャでの実行をサポートしています。
- ソフトウェアは、複数の Windows システム バージョンの x64 システムでテストされており、正常に安定して動作します。
